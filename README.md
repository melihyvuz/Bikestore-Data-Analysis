📊 Bikestore Data Analysis Project (Kaggle)
This project focuses on analyzing a bike store's sales, stock, and personnel performance using SQL and Python. It covers the entire data pipeline from database modeling to advanced visualization.

🇹🇷 Proje Özeti
Bu proje, bir bisiklet mağazasının satış, stok ve personel performansını SQL ve Python kullanarak analiz etmeyi amaçlamaktadır. Veritabanı modellemesinden ileri seviye görselleştirmeye kadar tüm veri süreçlerini kapsamaktadır.

🛠️ Technologies Used / Kullanılan Teknolojiler
SQL: Data extraction and business question analysis. / Veri çekme ve iş soruları analizi.

Python: Data cleaning and advanced visualization. / Veri temizleme ve ileri seviye görselleştirme.

Libraries: Pandas, Seaborn, Matplotlib.

Database: MySQL / SQLite.

📂 Project Structure / Proje Yapısı
/sql: SQL queries for turnover, stock, and performance analysis. / Ciro, stok ve performans analizi için SQL sorguları.

/visuals: Database schema and analysis charts. / Veritabanı şeması ve analiz grafikleri.

/notebooks: Python (Pandas & Seaborn) analysis files. / Python analiz dosyaları.

🔍 Key Insights from SQL / SQL Analizinden Temel Bulgular
The analysis performed via SQL queries answers the following business questions:
SQL sorguları ile aşağıdaki iş sorularına yanıt aranmıştır:

Brand Turnover: Calculating total revenue generated by each brand. / Marka bazlı ciro analizi.

Stock Alerts: Identifying products with critical stock levels (less than 5). / Kritik stok seviyesindeki ürünlerin tespiti.

Customer Engagement: Checking the customer-to-order ratio. / Müşteri sipariş etkileşim analizi.

Staff Performance: Ranking personnel based on total orders handled. / Personel satış performansı sıralaması.

Category Popularity: Finding the most demanded product categories. / Kategori bazlı popülerlik analizi.
